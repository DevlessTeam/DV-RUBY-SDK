# DVRUBYSDK

Official Devless Ruby SDK

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'DV_RUBY_SDK'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install DV_RUBY_SDK
